Wordpress Plugin Name: Woo Recent Posts
Plugin URI: http://www.jdstudios.us/plugins/woo-recent-posts
Description: Displays a list of recent posts using thumbnails in your woo theme. Filter by category, set the number to display, and set the woo thumbnail size.
Author: JD Studios (James Dudley)
Stable tag: 1.0.0


INSTALLATION INSTRUCTIONS:
1. copy the "woo-recent-posts" folder contained in this zip file into your wordpress plugins directly (usually /wp-content/plugins/)
2. login to your wordpress admin and go to "Plugins"
3. find "Woo Recent Posts" in the list, and click the Activate link


HOW TO USE:
- go to your Widgets page in the Wordpress admin (under "Appearance")
- drag the "Woo Recent Posts" widget from the available area into any of your sidebar(s)
- set your configuration as desired in the widget options panel
